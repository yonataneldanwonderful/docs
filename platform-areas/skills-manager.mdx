## Skills Manager

Skills Manager is the interface for creating, managing, and organizing skills across your organization.

## Features

- **Skill creation** — Build new skills with prompts and tools
- **Skill library** — Browse and reuse existing skills
- **Organization** — Organize skills by category or domain
- **Versioning** — Version skills for safe iteration
- **Sharing** — Share skills across agents and teams
- **Testing** — Test skills in isolation
- **Analytics** — Track skill usage and performance

## Use Cases

Use Skills Manager to:
- Create reusable skill components
- Organize skills by domain
- Share skills across agents
- Version and iterate on skills
- Monitor skill performance

Skills Manager enables efficient skill development and reuse.
