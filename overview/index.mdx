## Platform capabilities

From design to deployment to optimization, Wonderful provides everything you need to build agents at scale.

### Build and configure

| Goal                          | Guide                                                                      | Description                                                            |
| ----------------------------- | -------------------------------------------------------------------------- | ---------------------------------------------------------------------- |
| Create agents                 | [Agents](/core-concepts/agents)                                            | Build AI agents with system prompts, tools, and skills                 |
| Write agent prompts           | [Agents](/core-concepts/agents)                                            | Define persona, goals, and constraints for agent behavior              |
| Configure dynamic skills      | [Skills](/core-concepts/skills)                                            | Create modular competencies that load on demand                        |
| Connect tools and APIs        | [Tools](/core-concepts/tools)                                              | Enable agents to call APIs and perform actions                         |
| Set up deterministic tags     | [Tags](/core-concepts/tags)                                                | Create rules-based tags for consistent behavior tracking               |
| Configure voice settings      | [Agents](/core-concepts/agents)                                            | Set voice selection, speed, and style for voice interactions           |
| Manage agent versioning       | [Agents](/core-concepts/agents)                                            | Version agents with draft, versioned, and published states             |
| Set up base skills            | [Skills](/core-concepts/skills)                                            | Configure always-loaded skills for foundational capabilities           |
| Configure tenant tools        | [Tools](/core-concepts/tools)                                              | Share tools across multiple agents within a tenant                     |
| Control PII reduction         | [Agents](/core-concepts/agents)                                            | Configure privacy controls for sensitive data                          |

## Architecture

The Wonderful platform coordinates 6 core components:

1. **Agents** orchestrate conversations and decision-making across all channels

2. **Skills** provide modular competencies that load dynamically based on context

3. **Tools** enable deterministic API actions and capabilities

4. **Events** track all system activity for deterministic behavior and tagging

5. **Tags** annotate conversations for analytics, routing, and product logic

6. **Channels** deliver the experience across voice, chat, email, and WhatsApp with unified agent logic
