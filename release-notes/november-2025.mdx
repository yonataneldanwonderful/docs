## Agent Studio enhancements

- **New RAG management** - Create, attach, and detach RAGs directly within Agent Studio
- **Version comparison** - Compare different versions of agent instructions side-by-side
- **Improved layout** - Enhanced responsiveness and smoother animations across all screen sizes
- **Global variables support** - Added support for global variables in Agent Studio components
- **Collapsible sections** - New collapse/expand functionality in the prompt editor

## Voice & telephony

- **Deepgram integration** - Added Deepgram as a new Text-to-Speech (TTS) and Speech-to-Text (STT) provider
- **Improved call flow** - Enhanced call handling with better state management for forwards and hangups
- **Live transcripts fix** - Fixed transcript ordering issues and improved real-time transcript reliability

## UI/UX improvements

- **Auto-scroll fix** - Fixed auto-scroll behavior for completed calls in the transcript view
- **Better WebSocket management** - Improved real-time updates and connection handling

## Bug fixes

- **Deepgram TTS stability** - Fixed race conditions and memory leaks in the Deepgram synthesizer
- **Kafka consumer fix** - Resolved message ordering issues in the event consumer
- **S3 file processing** - Fixed file processing issues with S3 storage
- **WhatsApp dynamic skills** - Fixed dynamic skill functionality for WhatsApp channel
- **Invalid skill names** - Resolved bug with invalid skill name handling
- **Sub-RAG visibility** - Fixed issue where sub-RAG outputs weren't visible in Meta RAG configurations
